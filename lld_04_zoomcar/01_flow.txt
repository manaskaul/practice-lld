Requirements -

User can search for a Loaction they are in to find cars
User can search for the car they are intreseted in
System should be able to handle different types of Vehicles
System should charge the User based on the vehivle type and rental period
System should generate the invoice after the User has selected the vehicle and rental period
User pays the charge when the vehicle is rented


Functional Requirements -

System should allow users to search for available vehicles based on store location (branch)
System should allow users to filter vehicle by type (sedan/suv)
System should allow users to specify the rental period (interval of 1hr - start & end time)
System should show only vehicles that are available for the selected location and rental time range.
System should allow user to reserve the available vehicle for the selected time range
System should calculate the rental charge based on the vehicle type and the rental period
System should generate invoice and allow user to pay at the time of booking
System should allow vehicle pickup and return, hence updating the vehicle availability accordingly

Optional (won't do):
System should allow user to pay at the time of return
System should allow user to cancel before pickup
